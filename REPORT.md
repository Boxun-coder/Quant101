# REPORTING_INSTRUCTIONS.md

## Objective
Your final task is to build `src/report_generator.py`. This module must ingest the logs, metrics, and equity curves generated by the backtester, and automatically generate a professional, multi-page PDF report. 

## 1. Required Libraries
You will need to install the following (update `requirements.txt`):
* `matplotlib` and `seaborn` (for visualizations)
* `weasyprint` (for HTML to PDF conversion)
* `jinja2` (optional, for cleanly injecting data into the HTML)

## 2. The Report Philosophy & Content
The generated PDF MUST include the following sections. Do not skip any of these narratives:

* **Executive Summary & Philosophy:** Explain *why* the strategy works. State that it is a market-neutral statistical arbitrage strategy relying on transient mispricings (mean reversion, momentum) while stripping out systematic market risks.
* **Data Partitioning (IS vs OOS):** Explicitly break down the dates.
    * **In-Sample (2016-01 to 2020-12):** Used for factor discovery and tuning.
    * **Out-of-Sample (2021-01 to 2024-12):** The blind test. Emphasize that no parameter tweaking occurred here.
* **Backtest Assumptions:** * **T+1 Execution:** State that signals generated at the close of Day T are executed at the close of Day T+1 to eliminate look-ahead bias.
    * **Transaction Costs:** State the explicit deduction of 5 bps (0.05%) per unit of turnover.
    * **Slippage & Shorting:** Assume 100% borrow availability for the S&P 500 universe and negligible market impact due to large cap liquidity.

## 3. Required Plots
Your script must use `matplotlib` to generate the following plots, save them to memory (`io.BytesIO`), encode them as base64, and embed them into the HTML:
1.  **Cumulative Returns (IS vs OOS):** A single line chart showing the equity curve. Use a vertical dashed line to mark `2021-01-01`. Use blue for IS and orange for OOS.
2.  **Drawdown Curve:** A filled area chart showing the peak-to-trough drawdowns over time.
3.  **Factor Attribution (Optional but recommended):** A bar chart showing the individual Sharpe ratios of the Reversal, Volume, and Momentum factors independently.

## 4. Technical Workflow
1. Load `results_matrix.parquet` outputted by the backtester.
2. Calculate the Key Performance Indicators (Annual Return, Volatility, Sharpe, Drawdown, Turnover) separately for the IS and OOS periods.
3. Generate the matplotlib plots and base64 encode them.
4. Inject the metrics and base64 strings into an HTML template (using inline CSS to make it look professional, similar to institutional quant tear sheets).
5. Run `from weasyprint import HTML; HTML(string=html_string).write_pdf("Final_Report.pdf")`.
